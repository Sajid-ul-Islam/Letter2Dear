# 💌 ToMyInfida: Premium Love Letter SaaS

A feature-rich, modular, and privacy-focused web application for creating and sharing beautiful, interactive love letters. Designed for modern romance with a mobile-first approach.

## ✨ Premium Features

- **💌 Advanced Envelope Animation:** Realistic 3D unfolding interaction.
- **📸 Aesthetic Polaroid:** Attach memories via file upload or URL.
- **📅 Unlock Timers:** Set a specific date for the letter to reveal itself.
- **🎵 Music Boutique:** Integrated romantic background tracks.
- **🖋️ Custom Handwriting:** Choose from multiple elegant script fonts.
- **🎟️ Love Coupons:** Digital gift vouchers tucked inside every letter.
- **🌸 Atmosphere Effects:** Falling petals, soft rain, twinkling stars, and more.
- **🔐 Privacy First:** Client-side password protection.
- **🚀 Zero-Backend SaaS:** All data is encoded into the URL—no database needed, completely free to host.
- **📱 PWA Ready:** Installable on Android/iOS and optimized for the Google Play Store.

## 🛠️ Technology Stack

- **Frontend:** Vanilla HTML5, CSS3 (Glassmorphism), ES6+ JavaScript.
- **Shortener:** Integrated `is.gd` API for clean, shareable links.
- **Hosting:** Optimized for Surge.sh / Fly.io.

## 🚀 One-Click Deployment

This project is configured for rapid deployment to Surge.

### Prerequisites
1. Install Surge globally:
   ```powershell
   npm install -g surge
   ```

### Deploy to Live
Run the following command to push updates to **letter2dear.surge.sh**:
```powershell
npm run deploy
```

## 📁 Directory Structure
- `/css`: Modular styling (core, themes).
- `/js`: Business logic and effects.
- `index.html`: Main application entry.
- `sw.js`: Service Worker for PWA support.
